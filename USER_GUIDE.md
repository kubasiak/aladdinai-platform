# AladdinAI Platform - User Guide

## 🚀 Getting Started

### For New Users

**1. Create Your Account**
- Visit: https://ai-webpages.web.app/register.html
- Enter your email and password
- Click "Create Account"
- You'll see a success message with your account ID

**2. Login**
- Click "Go to Login" or visit: https://ai-webpages.web.app/login.html
- Enter your email and password
- Click "Login"

**3. Choose Your Template**
- First-time login will take you to template selection
- You'll see two options:

  **💎 Classic Single Card**
  - Simple, elegant single-card layout
  - Hero section with company name & tagline
  - About section with two text blocks
  - Contact section with email, phone & address
  - Perfect for simple landing pages

  **🤖 AImachine Professional**
  - Professional multi-section website
  - Fixed navigation header
  - Dual hero sections with background images
  - Company showcase
  - Solutions grid (4 service cards)
  - Project portfolio (4 images)
  - Call-to-action section
  - Multi-column footer
  - Perfect for tech companies and service providers

- ⚠️ **Important:** This choice is permanent and cannot be changed later
- Click your preferred template
- Click the continue button

**4. Customize Your Page**
- You'll be taken to the admin panel
- Edit your content, upload media, adjust styling
- Click "💾 Save All Changes"
- Your page is now live!

**5. View Your Public Page**
- Your page URL is shown in the admin panel
- Share this URL with anyone
- It's publicly accessible

## 📝 Using the Admin Panel

### Content Editing
- Click on any text to edit it directly
- Changes are saved when you click "💾 Save All Changes"

### Media Library
- **Upload Videos:** Background videos for your page
- **Upload Audio:** Background music/audio
- **Screenshots:** Auto-generated from videos for quick loading
- **Card Background:** Choose a background image for your card

### Styling Controls
- **Font Size:** Adjust title and body text sizes
- **Colors:** Change text colors
- **Transparency:** Control overlay darkness
- **Animation:** Adjust fade animation speed
- **Video Speed:** Control playback speed

### Import/Export
- **📥 Export Library:** Download all your media to a file
- **📤 Import Library:** Upload media from another project
- Useful for sharing media between different pages

## 🎯 Quick Tips

### For Best Results

**Images:**
- Use high-quality images (1920x1080 recommended)
- Compress images before uploading
- Use relevant, professional photos

**Content:**
- Keep text concise and clear
- Use compelling headlines
- Include clear calls-to-action

**Styling:**
- Test on mobile devices
- Ensure text is readable
- Balance transparency and visibility

**Performance:**
- Limit video file sizes (under 15MB)
- Optimize images before upload
- Use compressed audio files

## 🔧 Common Tasks

### Change Your Company Name
1. Login to admin panel
2. Click on company name text
3. Type your new name
4. Click "💾 Save All Changes"

### Add a Background Video
1. Go to Media Library section
2. Click "📤 Upload" in Videos section
3. Select your video file (MP4, under 15MB)
4. Wait for upload
5. Click "Use" on the uploaded video
6. Click "💾 Save All Changes"

### Update Contact Information
1. Click on email/phone/address text
2. Edit directly
3. Click "💾 Save All Changes"

### Export Your Media
1. Scroll to Media Library
2. Click "📥 Export Library"
3. Save the JSON file
4. Use this file to import media to another project

## 🆘 Troubleshooting

### "Cannot change template" message
- Template choice is permanent by design
- Create a new account if you want a different template

### Images not loading
- Check file size (under 15MB for videos, 5MB for images)
- Ensure file format is supported (JPG, PNG, MP4, WebM)
- Try refreshing the page

### Changes not appearing
- Make sure to click "💾 Save All Changes"
- Clear browser cache and refresh
- Check browser console for errors (F12)

### Login issues
- Verify email and password are correct
- Check for typos
- Password must be at least 6 characters

## 🌐 Your Public Page

### Sharing Your Page
- Your page URL is shown in the admin panel
- Format: `https://ai-webpages.web.app/your-slug`
- Share this URL with anyone
- No login required to view

### Custom URL Slug
- Set in admin panel under "Your Public URL"
- Use lowercase letters, numbers, and hyphens
- Keep it short and memorable
- Example: `my-company` → `ai-webpages.web.app/my-company`
